Mit der „Kommentar-Box” können Besucher Kommentare auf
der Seite hinterlassen.

 - In Beliebig viele Seiten kann die „Kommentar-Box” eingebunden werden.
 - Die Kommentare werden in einer MySQL-Datenbank gespeichert.
 - Der Admin. erhält eine E-Mail bei einem Kommentar.
 - Die Kommentare können frei gegeben, geändert oder gelöscht werden.
 - Sicherheitsabfrage (Spamschutz).
 - Eine Bewertungsfunktion für die Seite.
 - Smileys werden dargestellt.
 - Aufteilung der Kommentare auf mehrere Seiten (Blättersystem).
 - Die „Kommentar-Box” kann über eine externe CSS-Datei angepasst werden.


Funktionsweise:
 Der Besucher kann einen Kommentar eintragen, wurde ein Eingabefehler entdeckt, 
 wird der Besucher darauf hingewiesen.

 Nachdem der Besucher seinen Kommentar erfolgreich eingetragen hat, 
 wird der Admin. über E-Mail benachrichtigt. 
 Zum freischalten des Kommentars klickt dieser nun auf den in der E-Mail enthaltenen Link
  (den Link eventuell kopieren und in die Adresszeile des Browsers einfügen).

 Es öffnet sich ein Browser-Fenster mit dem Formular. Hier kann der Admin. den Kommentar 
 gegebenfalls editieren (oder etwas dazu schreiben), Benutzername und Passwort eingeben 
 und auf "Ausführen" klicken.


Installation - Führen Sie folgende Anweisungen nacheinander aus:
 1. In der Datei: "kommentar.php" passen Sie die Einstellungen an.
 2. Laden Sie alle Dateien auf Ihrem Webserver hoch.
 3. Rufen Sie die Datei: "install.php" in Ihrem Browser auf.
 4. Geben Sie Ihren Benutzername und das Passwort ein und drücken auf dem Button "Ausführen".
 5. Nach erfolgreicher Installation, sollten Sie die Datei: "install.php" vom Webserver entfernen.


Das Einbinden der „Kommentar-Box” in eine Seite ist denkbar einfach:
<?php include "kommentar.php";?>

CSS einbinden:
<link rel="stylesheet" href="kommentar.css">

 - In der Datei: "demo.php" finden Sie eine Demonstration dazu.


Tipp:
Möchten Sie nachträglich einen Kommentar bearbeiten, so fügen Sie in der
Adresszeile des Browsers folgendes ein: dateiname.php?aktion=bearbeiten
